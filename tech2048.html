Description

It test the speed, accuracy and basic logical ability of the participant.
The game is mainly aimed at newbies to programming who basically know C language and programming. The game is modelled in the form of a 2048 gridd. Each grid has a hidden question at the back with 2 points each. Consecutive grids of the same number can be combined together so that the empty slot fills up with C new question . The more the participant solves and joins the grids. The more he combines , the more hehas an ability to get new questions and score more. The questions are based on 2 simple C and aptitude. The time limit for the game is about 5 minutes and the points are counted for solved questions.
